Environment Setup

All labs require Python 3.9+ and the following packages. Install once and all labs will work.

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu pip install transformers datasets accelerate pip install scikit-learn matplotlib numpy librosa pip install soundfile tqdm

GPU optional but recommended for speech labs. All text labs run comfortably on CPU in <5 minutes.
